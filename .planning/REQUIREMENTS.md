# Requirements

## 플랫폼
- **Framer** (디자인 + CMS + 호스팅 올인원)

---

## 필수 기능 (Must Have)

### 포트폴리오 쇼케이스 페이지
- [ ] 프로젝트 목록 그리드/갤러리 레이아웃
- [ ] 카테고리 필터 (광고소재 / 프로모션 / 패키징·신제품 / 웹사이트 / 오프라인 팝업)
- [ ] 프로젝트 썸네일 이미지 표시
- [ ] 프로젝트 제목 + 카테고리 표시

### 프로젝트 상세 페이지
- [ ] 프로젝트 배경/개요 텍스트
- [ ] 작업 과정 설명
- [ ] 결과물 이미지 갤러리 (다수 이미지 업로드 가능)
- [ ] 이전/다음 프로젝트 네비게이션

### About 페이지
- [ ] 자기소개 텍스트
- [ ] 프로필 사진
- [ ] 보유 스킬/역량 표시
- [ ] 연락처 (이메일 등)

### CMS (관리자 기능)
- [ ] Framer CMS에서 프로젝트 추가/수정/삭제
- [ ] 이미지 업로드
- [ ] 카테고리 지정
- [ ] 공개/비공개 설정

### 기술 요건
- [ ] 모바일 반응형 (Framer 자동 처리)
- [ ] 커스텀 도메인 연결 가능
- [ ] 페이지 로딩 속도 적정 수준

---

## 선택 기능 (Nice to Have)

- [ ] 홈 화면 인트로 애니메이션
- [ ] 프로젝트 호버 인터랙션 효과
- [ ] 비밀번호 보호 페이지 (특정 클라이언트 작업물)
- [ ] 방문자 문의 폼 (Contact)
- [ ] OG 이미지 설정 (SNS 공유 시 썸네일)

---

## 범위 외 (Out of Scope)

- 블로그/글쓰기 기능
- 다국어(영문) 지원
- 방문자 통계 대시보드
- 결제 기능
