# 브랜드 디자이너 포트폴리오 사이트

## What This Is

3년차 브랜드 디자이너의 취업/이직용 포트폴리오 사이트. 브랜드 인하우스 면접관을 대상으로, 광고소재·프로모션·패키징·웹사이트·오프라인 팝업 등 다양한 작업물을 균형 있게 보여준다. 관리자(본인)가 코드 없이 작업물을 직접 업로드·수정할 수 있는 CMS 기능을 포함한다.

## Core Value

면접관이 사이트를 열었을 때, 디자이너의 역량과 작업 범위를 한눈에 파악할 수 있어야 한다.

### Active

### Out of Scope

- 다국어(영문) 지원 — 우선 국내 취업 활동 중심, 추후 필요 시 추가
- 블로그/글쓰기 기능 — 포트폴리오 집중
- 방문자 통계 대시보드 — 복잡도 대비 필요성 낮음

## Context

- 사용자(디자이너 본인)는 코딩 경험 없음 → 코드 수정 없이 운영 가능해야 함
- 작업물 카테고리: 광고소재, 프로모션, 신제품 디자인(패키징/촬영/상세페이지), 웹사이트 디자인, 오프라인 팝업
- 디자인 방향 미정 → UI 레퍼런스 탐색 단계 필요
- 브랜드 인하우스 면접관 대상 → 전문성·완성도 중시

## Constraints

- **기술 역량**: 코딩 경험 없음 — 운영은 반드시 노코드/CMS로 해결
- **비용**: 무료 또는 저비용 호스팅 선호 (명시되지 않았으나 일반적 가정)
- **유지보수**: 혼자서 작업물 추가/수정 가능해야 함
