# Roadmap

## 프로젝트: 브랜드 디자이너 포트폴리오 사이트 (Framer)

---

## Phase 1 — Framer 환경 세팅 & 템플릿 선택
**목표:** Framer 계정 생성, 적합한 템플릿 선택, CMS 구조 세팅

### Plans
1. **Framer 가입 및 기본 설정**
   - framer.com 계정 생성
   - 무료 플랜 시작
   - 포트폴리오 전용 템플릿 탐색 (Gibson, Hanssen, Viper 등)
   - 템플릿 선택 기준: CMS 지원 여부, 갤러리 레이아웃, 미니멀 디자인

2. **CMS 컬렉션 구조 설계**
   - "Projects" 컬렉션 생성
   - 필드 정의: 제목, 카테고리, 썸네일 이미지, 상세 이미지들, 배경 설명, 작업 내용, 공개 여부
   - 카테고리 옵션 설정: 광고소재 / 프로모션 / 패키징·신제품 / 웹사이트 / 오프라인 팝업

**완료 기준:** Framer 프로젝트 생성 완료, CMS 컬렉션 세팅 완료

---

## Phase 2 — 포트폴리오 콘텐츠 정리 & 입력
**목표:** 작업물 4~7개 정리 후 CMS에 입력

### Plans
1. **작업물 선별 및 정리**
   - 각 카테고리별 대표 프로젝트 1~2개 선택
   - 프로젝트별 이미지 수집 및 리사이징 (웹 최적화)
   - 프로젝트별 설명 텍스트 초안 작성 (배경, 내 역할, 결과)

2. **CMS 콘텐츠 입력**
   - 프로젝트 4~7개 CMS에 입력
   - 썸네일 + 상세 이미지 업로드
   - 카테고리 태그 지정

**완료 기준:** 모든 프로젝트 CMS 입력 완료, 쇼케이스 페이지에서 정상 표시

---

## Phase 3 — 페이지 디자인 커스터마이징
**목표:** 템플릿을 본인 브랜드에 맞게 커스터마이징

### Plans
1. **홈/쇼케이스 페이지 커스터마이징**
   - 컬러 팔레트 설정 (본인 브랜드 톤)
   - 타이포그래피 선택 (한글 폰트 포함)
   - 헤더/네비게이션 구성
   - 프로젝트 카드 레이아웃 조정

2. **프로젝트 상세 페이지 커스터마이징**
   - 이미지 갤러리 레이아웃 설정
   - 텍스트 영역 스타일 조정
   - 이전/다음 프로젝트 네비게이션 추가

3. **About 페이지 제작**
   - 프로필 사진 + 자기소개 텍스트
   - 스킬/역량 섹션
   - 연락처 정보

**완료 기준:** 모든 페이지 디자인 완성, 브랜드 톤 통일

---

## Phase 4 — 모바일 최적화 & 퍼블리시
**목표:** 모바일 뷰 확인, 최종 점검 후 배포

### Plans
1. **모바일 반응형 점검**
   - Framer 모바일 뷰에서 각 페이지 확인
   - 레이아웃 깨지는 부분 수정
   - 이미지 로딩 속도 확인

2. **최종 점검 및 배포**
   - 전체 페이지 링크/기능 테스트
   - SEO 기본 설정 (페이지 제목, 설명)
   - Framer 서브도메인으로 1차 배포
   - (선택) 커스텀 도메인 연결 ($10/월 Basic 플랜)

**완료 기준:** 사이트 라이브, 모바일/데스크탑 모두 정상 작동
